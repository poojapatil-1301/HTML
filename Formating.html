This is Bold Tag.
Syntax is :- "< b >< /b>"
This tag elements defines bold text without any importance.

This is Strong/Important Tag.
Syntax is :-"< strong >< /strong>"
This tag elements defines strong text with added semantic "Strong" importance.

This is Italic Tag.
Syntax is :- "< i >< /i>"

This is Emphasized Tag.
Syntax is :- "< em >< /em>"

This is Super Script Tag.
Syntax is :-"< sup >< /sup>"
Example :- I'm MCA 3rd Year Student.

This is SubScript Tag.
Syntax is :-"< sub >< /sub>"
Example :- Log(102)

This is Inserted Tag.
Syntax is :- "< ins >< /ins>"

This is Delete Tag.
Syntax is :- "< del >< /del>"

This is Mark Tag.
Syntax is :- "< mark >< /mark>"
